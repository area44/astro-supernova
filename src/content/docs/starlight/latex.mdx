This guide shows you how to render LaTeX math formulas in your Starlight project using the built-in math support in [`Sätteri`](https://satteri.bruits.org).

## Step-by-step Guide

<Steps>

1. **Install the required dependencies**

   Ensure you have `satteri` and the Astro Sätteri integration installed:

   <PackageManagerCommand command="add satteri @astrojs/markdown-satteri" />

2. **Configure Sätteri processor in `astro.config.mjs`**

   Open your `astro.config.mjs` file and enable the `math` feature in the Sätteri processor:

   ```js
   // astro.config.mjs
   import { satteri } from "@astrojs/markdown-satteri";
   import { defineConfig } from "astro/config";

   export default defineConfig({
     markdown: {
       processor: satteri({
         features: {
           math: true,
         },
       }),
     },
   });
   ```

3. **Style the math output (Optional)**

   By default, Sätteri renders inline math as `<code class="language-math math-inline">` and display math as `<pre><code class="language-math math-display">`. You can style these classes in your `src/styles/global.css` or use a client-side library like KaTeX or MathJax to render them into beautiful formulas.

</Steps>

## Example Usage

Create a file like `src/content/docs/example.mdx` and add:

```mdx
When $a\ne0$, there are two solutions to $ax^2 + bx + c = 0$ and they are

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

The code above generates the following on the page:

When $a \ne 0$, there are two solutions to $ax^2 + bx + c = 0$ and they are

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
