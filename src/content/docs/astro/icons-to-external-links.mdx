This guide shows how to append an icon to the end of each external link, making it clear to users that the link will take them away from your site.

## Step-by-Step Guide

<Steps>

1. Install the plugin  
   Add the [`rehype-external-links`](https://github.com/rehypejs/rehype-external-links) and [`@types/hast`](https://github.com/DefinitelyTyped/DefinitelyTyped/tree/master/types/hast) packages:

   <PackageManagerCommand command="add rehype-external-links @types/hast" />

3. Configure Astro to use the plugin  
   Open your `astro.config.mjs` and add the plugin to the Markdown config:

   ```js ins={3, 7}
   // astro.config.mjs
   import { defineConfig } from "astro/config";
   import { rehypeExternalLinks } from "./src/plugins/rehype-external-links";

   export default defineConfig({
     markdown: {
       rehypePlugins: [rehypeExternalLinks()],
     },
   });
   ```

4. Add custom styles for the external link icon in your global CSS:
   ```css
   /* Prevent style conflicts by targeting only external link icons */
   a[href^="http"] .external-link-icon {
     display: inline-block;
     width: 0.75em;
     height: 0.75em;
     margin-left: 0.2em;
     vertical-align: text-bottom;
     fill: currentColor;
   }
   ```

</Steps>

## Example Usage

Create or open a Markdown or MDX file, such as `src/content/docs/example.mdx`, and add:

```mdx
[Google](https://google.com)
```

When rendered, the external link will include a small arrow icon at the end, while internal links remain unchanged:

[Google](https://google.com)
