Supernova supports both technical recipes and narrative blog posts. While recipes focus on a objective "how-to" approach, blog posts allow for a more conversational and subjective exploration of topics.

## Formatting Guidelines

Blog posts should be stored in `src/content/docs/blog/` and follow these key principles:

- **Tone**: Professional, conversational, and personal.
- **Visuals**: Use diagrams and video embeds to break up text and explain complex ideas.
- **Headers**: Use clear and descriptive headings.
- **Engagement**: Share personal insights and "lessons learned" during the development process.

## Step-by-Step Guide

<Steps>

1.  **Select a Topic**: Choose a theme or narrative that would be better suited for a blog format than a recipe.
2.  **Location**: Create a new `.mdx` file in `src/content/docs/blog/`.
3.  **Define Frontmatter**:
    - Set the `title`, `description`, and `publishDate`.
4.  **Enrich Content**:
    - Use the `<YouTube />` component to embed relevant video content.
    - Use `<Mermaid />` for diagrams.
    - Use standard Starlight components as needed.
5.  **Review**: Ensure the post reads as a cohesive narrative.

</Steps>
